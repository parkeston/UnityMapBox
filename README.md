# UnityMapBox
Simple unity game that uses Mapbox SDK to create real world based game environments.
### Player 1 controls
- WASD - movement;
- Mouse - look;
- Q - invisibility.

### Player 2 controls
- Arrows - movement;
- Enter - shoot.
### Goals
- Player's 1 - collect green light beams;

- Player's 2 - kill player 1.
